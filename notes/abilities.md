# 能力値に関するノート

馬の能力値に関して

- 速度系
- 体力系
- 器用系
- 性質系

## 1. 速度系（Speed & Acceleration）

現在はギアが2段階、速度が3種類にしている。
中央・地方の1600m〜2000mの走破タイム（1600mで約96秒〜105秒＝秒速15.2m〜16.6m）と、上がり3ハロン（約34秒〜40秒＝秒速15.0m〜17.6m）の逆算から導き出せるリアルな数値レンジです。

- start_speed: 12.0 〜 14.5＞m/s / ゲートを出てから最初の数秒間の速度。
- cruise_speed: 15.0 〜 16.5＞m/s / 道中の巡航速度。16.0m/s（時速57.6km）が標準的な基準
- spurt_speed: 17.0 〜 18.5＞m/s / 直線最高速度。トップクラスの末脚で18.5m/s（時速66.6km）

- start_acceleration: 2.5 〜 3.5＞m/s² / ゼロ発進時の加速力
- cruise_acceleration: 0.4 〜 0.6＞m/s² / 道中の微調整用（spurtの50%前後）。
- spurt_acceleration: 0.8 〜 1.2＞m/s² / スパートをかけてから最高速に達するまでのトガリ度
- spurt_endurance: 400 〜 700＞m / スパートを維持できる距離（または20〜40秒の時間）
- top_speed_potential: 18.0 〜 19.0＞m/s / 下り坂や追い風、極限状態でも絶対に超えない物理限界値

## 体力系（Stamina & Efficiency）

スタミナの絶対値そのものは物理エンジンの内部フレーム数（Update回数）に依存するため、ここでは「良馬場の1600mを走りきれる消費量を1.0（基準）」とした場合の係数レンジとして提案します

- total_stamina: 1600 〜 2400＞pt / 距離（m）をベースに、精神力や総量として持たせる基礎スタミナ
- stamina_waste_rate: 0.95 〜 1.05＞倍率 / 標準を 1.0 とし、気性難やロスで消費が増える馬は 1.05
- heavy_track_aptitude: 0.90 〜 1.10＞倍率 / 重・不良馬場時の消費ペナルティ。適性高＝0.92（低消費）、適性低＝1.10
- weight_tolerance: 0.98 〜 1.02＞倍率 / 斤量1kg増減ごとのスタミナ（または加速）へのデバフ影響度
- distance_flexibility: 0.85 〜 1.00＞倍率 / 適性距離外に出走した際の、最大スタミナ（total_stamina）の減衰率

## 器用系（Agility & Adaptability）

- cornering_ability: 0.85 〜 0.98＞倍率 / カーブ時の速度維持率。小回り地方競馬ではこの差が大きく響く。コーナーを曲がる能力
- gate_reaction: -0.20 〜 +0.20＞秒 / ゲートが開いてから start_speed に移行するまでのタイムラグ。スタート時の反応力
- stability_factor: 0.01 〜 0.05＞乱数幅 / 毎フレーム（または毎秒）の巡航速度の揺らぎ（ノイズ）の大きさ。安定性
- base_agility: 1.0 〜 2.5＞m/s / レーンを横に移動する際の横軸の最大移動速度。俊敏さ。機動性能
- lane_change_frequency: 0.05 〜 0.20＞確率 / 前方が詰まっていない平時に、レーンを動こうとする1秒あたりの確率。レーン移動頻度
- prefers_inside: 0.10 〜 0.80＞確率 / 1〜3枠を引いた際、通常より前目のポジション（1・2番手）を取りにいく確率。内枠時の先行率
- pace_switching_agility: 0.5 〜 1.5＞秒 / 前方のペース変化を検知してから、自身の加速・減速を開始するまでのラグ
- course_cornering_efficiency: 0.95 〜 1.05＞倍率 / 特定の競馬場（右回り・左回り、大井などの大井特有の砂）への乗算補正

## 性質系（Temperament & Strategy）

- strategy: 0, 1, 2, 3（Lead,Stalker,Closer,Rear）＞ID / 0:逃げ、1:先行、2:差し、3:追込 の固定フラグ。脚質
- pacing_strategy_bias: 0.05 〜 0.95＞位置割合 / 「全出走頭数の前方から何％の位置をキープしたいか」（逃げなら0.05、追込なら0.90）
- grit_factor: 1.01 〜 1.04＞倍率 / 直線で他馬と横並び（例: 側対地距離0.5m以内）の時、一時的に上がる速度（またはスタミナ軽減）係数
- mental_stability: 0.95 〜 1.05＞倍率 / 馬体重の増減データから逆算。レース開始時に全能力の最大値を ±5% 程度ランダムに上下させる
